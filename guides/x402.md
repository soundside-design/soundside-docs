# x402: Pay-Per-Call with Crypto

x402 lets any agent call Soundside tools by paying with USDC on Base. No API key, no account, no signup.

## How It Works

1. Agent sends a `tools/call` request to `https://mcp.soundside.ai/mcp`
2. If no valid API key or payment is included, Soundside returns **HTTP 402** with payment requirements
3. Agent pays the exact USDC amount via the facilitator
4. Agent re-sends the request with the payment receipt
5. Soundside executes the tool and returns the result

## Discovery

```bash
curl https://mcp.soundside.ai/api/x402/status
```

Returns:
```json
{
  "enabled": true,
  "endpoint": "https://mcp.soundside.ai/mcp",
  "network": "eip155:8453",
  "token": "USDC",
  "facilitator_url": "https://api.cdp.coinbase.com/platform/v2/x402",
  "pricing_model": "ceiling-quote",
  "enabled_tools": [
    {"tool": "create_image", "provider": "vertex", "price_usdc": "0.10", "sync": true},
    {"tool": "create_video", "provider": "luma", "price_usdc": "1.00", "sync": false}
  ]
}
```

## Wallet Setup

You need:
- An Ethereum wallet with USDC on **Base mainnet** (chain ID 8453)
- The `x402` Python or TypeScript SDK

### Getting USDC on Base

1. Buy USDC on Coinbase
2. Send to your wallet on Base network
3. Or bridge from Ethereum mainnet via [Base Bridge](https://bridge.base.org)

Typical costs: `create_image` is $0.10-0.20, `create_video` is $0.50-1.00. A full pipeline (image → video → music → edit) runs about $2-3.

## Async Resources

Some tools (video, music) are async — they return a `resource_id` immediately, and the actual media is generated in the background.

To check status:
```
GET https://mcp.soundside.ai/api/x402/resource/{resource_id}
Header: x402-wallet: <your-wallet-address>
```

Returns:
```json
{
  "resource_id": "...",
  "state": "completed",
  "storage_url": "https://storage.googleapis.com/...",
  "mime_type": "video/mp4"
}
```

## Error Handling

| HTTP Code | Meaning |
|-----------|---------|
| 402 | Payment required — include x402 payment |
| 422 | Validation error (invalid provider, missing params) |
| 503 | Billing unavailable — retry |

## SDK Examples

### TypeScript (x402)

```typescript
import { createPublicClient, http } from 'viem';
import { base } from 'viem/chains';
import { paymentMiddleware } from 'x402-axios';

const client = axios.create();
client.interceptors.request.use(
  paymentMiddleware(walletClient, { network: 'eip155:8453' })
);

// Now any 402 response is automatically paid
const response = await client.post('https://mcp.soundside.ai/mcp', {
  jsonrpc: '2.0',
  id: '1',
  method: 'tools/call',
  params: {
    name: 'create_image',
    arguments: { prompt: 'A fox in autumn forest', provider: 'vertex' }
  }
});
```
